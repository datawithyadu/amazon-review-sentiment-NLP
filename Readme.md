# Amazon Review Sentiment Analysis (NLP)

## 📌 Project Overview
This project builds an end-to-end Natural Language Processing (NLP) pipeline to classify Amazon product reviews into sentiment categories. The model processes raw text data, applies preprocessing techniques, and uses machine learning to predict sentiment.

---

## 🚀 Objectives
- Clean and preprocess textual review data
- Convert text into numerical features using TF-IDF
- Train a machine learning classification model
- Evaluate performance using accuracy and other metrics

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- TF-IDF Vectorization
- Jupyter Notebook

---

## 🔍 Methodology

### 1️⃣ Data Preprocessing
- Removed special characters
- Converted text to lowercase
- Tokenization
- Stopword removal
- Stemming

### 2️⃣ Feature Engineering
- Applied **TF-IDF (Term Frequency – Inverse Document Frequency)** to convert text into numerical vectors

### 3️⃣ Model Training
- Trained a machine learning classifier for sentiment prediction

### 4️⃣ Evaluation
- Measured model performance using accuracy score
